## 🎯 Features

### Core Trading Tools
- ✅ **Trade Journal** - Log every trade with full details
- ✅ **Performance Analytics** - Charts, win rates, session analysis
- ✅ **Pattern Memory System** - Track which patterns work best
- ✅ **Risk Management** - Daily limits, stop-loss rules, position sizing
- ✅ **Strategy Builder** - Backtest custom strategies on your history

### Risk Controls
- 🛑 Auto-stop after 2 consecutive losses
- 🛑 Daily risk limit (max 5% of capital)
- 🛑 Pattern blocking under 60% win rate
- 🛑 Confidence-based signal evaluation

### Data Features
- 💾 Local storage persistence
- 📥 CSV export
- 📱 PWA - Installable & offline capable
